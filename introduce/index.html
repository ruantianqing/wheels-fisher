wheels-fisher

wheels-fisher UI 是一个实用的 UI 简易框架，目前提供了一些常用组件（不定期更新），适合 PC 端和移动端使用。

现已支持

Button（按钮）🎉 、Input（输入框）🎉 、Grid（栅栏）🎉 、Layout（布局）🎉 、Tabs（标签页）🎉 、Popover（弹出框）🎉 、Collapse（折叠面板）🎉

why wheels

wheels 就是轮子，这是我在使用 Vue 的过程中尝试实现的 UI 组件（造的轮子）集合，希望对你有用。
